## ðŸŽ¨ Design Philosophy

The interface is intentionally designed around the following principles:

- **Narrative over catalog**  
  Products are presented as part of a visual story instead of a standard grid-based layout.

- **Visual hierarchy & rhythm**  
  Uneven card sizing, staggered positioning, and spacing are used to guide attention and create interest.

- **Progressive disclosure**  
  Detailed product information is revealed via a bottom-sheet modal, keeping the main layout clean and focused.

- **Subtle interactions**  
  Hover states, transitions, and animations are intentionally minimal to enhance usability without distraction.
